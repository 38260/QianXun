# AI创作助手平台 - 产品需求文档（基础版）

## 1. 产品简介

### 1.1 产品概述
AI创作助手是一个帮助内容创作者获取灵感和生成创意方案的智能工具。通过抓取各大平台的热点数据，结合AI技术为用户提供个性化的内容创作建议。

### 1.2 目标用户
- 自媒体创作者
- 短视频制作者  
- 内容营销人员
- 个人博主

### 1.3 核心功能
- 实时热点数据展示
- 个性化用户画像构建
- AI创意内容生成
- 智能对话助手

## 2. 功能需求

### 2.1 主要功能模块

#### 2.1.1 热点数据模块
**功能描述**：展示各大平台的实时热点信息

**具体功能**：
- 热点话题列表展示
- 按平台分类筛选（微博、抖音、B站等）
- 热度排序和时间排序
- 热点详情查看
- 数据实时更新

#### 2.1.2 用户画像模块  
**功能描述**：帮助用户构建个人创作偏好画像

**具体功能**：
- 兴趣标签选择（科技、娱乐、教育等）
- 自定义标签添加
- 用户描述文本输入
- 画像预览和编辑
- 标签管理（添加/删除）

#### 2.1.3 AI生成模块
**功能描述**：基于用户画像和热点数据生成创作方案

**具体功能**：
- 创意标题生成
- 内容大纲生成  
- 多个方案对比
- 创作建议提供
- 方案收藏和历史记录

#### 2.1.4 AI对话模块
**功能描述**：提供智能对话助手功能

**具体功能**：
- 自然语言对话
- 创作问题咨询
- 热点分析解答
- 对话历史记录
- 上下文理解

#### 2.1.5 数据源模块
**功能描述**：展示平台数据来源和统计信息

**具体功能**：
- 各平台数据统计
- 数据更新时间显示
- 平台数据详情查看
- 数据质量监控

## 3. 技术实现

### 3.1 技术架构
- **前端**：Next.js + React + TypeScript + TailwindCSS
- **后端**：Python + Flask + 多个API服务
- **AI模型**：Ollama本地部署（Qwen2.5、DeepSeek-R1）
- **数据存储**：MySQL数据库 + JSON文件
- **数据采集**：Python爬虫 + 定时任务

### 3.2 系统架构
```
前端应用 ←→ API网关 ←→ 业务服务
    ↓           ↓         ↓
  用户界面   接口管理   数据处理
                        AI生成
                        数据采集
```

### 3.3 核心技术组件
- **数据采集引擎**：多平台爬虫系统
- **AI生成引擎**：本地大语言模型
- **数据处理引擎**：热点分析和用户画像
- **API服务层**：统一接口管理

## 4. 用户界面

### 4.1 主要页面
1. **主页**：热点数据展示，实时更新
2. **用户画像页**：标签选择和个人信息设置  
3. **AI分析页**：智能对话界面
4. **数据源页**：各平台数据统计和详情
5. **项目亮点页**：产品功能介绍

### 4.2 界面特点
- 简洁现代的设计风格
- 响应式布局，支持多设备
- 直观的操作流程
- 清晰的信息层级

## 5. 业务流程

### 5.1 用户使用流程
1. 访问平台查看热点数据
2. 构建个人用户画像
3. 基于画像生成创作方案
4. 通过AI对话获取更多建议
5. 查看数据源了解详细信息

### 5.2 数据处理流程
1. 定时抓取各平台热点数据
2. 数据清洗和结构化处理
3. 存储到数据库和文件系统
4. 通过API接口提供给前端
5. 实时更新和展示
